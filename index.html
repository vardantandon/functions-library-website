Complex Quizzes

This is a demo website which contains all the useful MATLAB functions that can be taken into account while making any quizzes.


  1. quiz_start() : This function has to be called before question_cloze or any other question generation function.

  2. quiz_end() : This function has to be called after the last question_cloze or any other question generation function.

  3. question_cloze(num, nameStr, questionStr, generalFeedbackStr, hint1, hint2, fileStrQuestion, fileStrFeedback) : Produces string output for a Cloze type question.
    Input parameters
    num - number: Question Number
    nameStr - string: Question Name
    questionStr - string: The content of the question, including output from cloze_mcq/cloze_numerical/cloze_short
    generalFeedbackStr - Feedback shown to student after question is completed
    hint1 - string: hint for the first incorrect attempt
    hint2 - string: hint for the second incorrect attempt
    fileStrQuestion - string: xml code that contains a file (eg an image) for the question section
    fileStrFeedback - string: xml code that contains a file (eg an image) for the feedback section

  4. cloze_mcq(questionType, correctNo, varargin) : Produces code to embed an multiple choice answer inside a Cloze type q.
    Input parameters
    questionType - 'drop', 'vertical', 'horizontal'
    correctNo - correct option no starting with 1
    varargin - 'Option 1', 'error message', 'Option 2', 'error message', ..

  5. cloze_short(answer, rightFeedback, wrongFeedback) : Produces code to embed a short answer inside a Cloze type q.
    Input parameters
    answer - number: correct answer
    rightFeedback - string: correct answer feedback
    wrongFeedback - string: wrong answer feedback

  6. cloze_numerical(answer, error, rightFeedback, wrongFeedback) : Produces code to embed a numerical answer inside a Cloze type q.
    Input parameters
    answer - number: correct answer
    error - number: acceptable error
    rightFeedback - string: correct answer feedback
    wrongFeedback - string: wrong answer feedback

  7. generateImageString(plot, imageNameStr) : Produces code to embed an image/plot within a question.
    Input parameters
    plot - plot: The plot to convert
    imageNameStr - string: name of image file

  8. arror(varargin) : Draws a line with an arrowhead and any other properties , if needed like length , start and end points etc.

  9. matrix_table(matrixName, matrix) : Matrix table code generator which produces the code for a matrix table with the brackets around it.
    Input parameters
    matrixName - string: the label for the matrix
    matrix - 2D strings array: 2D array containing the strings for each matrix cell
