Complex Quizzes

This is a demo website which contains all the useful functions that can be taken into account while making any quizzes.


  1. quiz_start() : This function has to be called before question_cloze or any other question generation function.
  2. quiz_end() : This function has to be called after the last question_cloze or any other question generation function.
  3. question_cloze(num, nameStr, questionStr, generalFeedbackStr) : Produces string output for a Cloze type question.
    Input parameters
    num - number: Question Number
    nameStr - string: Question Name
    questionStr - string: The content of the question, including output from cloze_mcq/cloze_numerical/cloze_short
    generalFeedbackStr - Feedback shown to student after question is completed
  4. cloze_mcq(questionType, correctNo, varargin) : Produces code to embed an multiple choice answer inside a Cloze type q
  5. cloze_short(questionType, correctNo, varargin) : Produces code to embed a short answer inside a Cloze type q
  6. cloze_numerical(questionType, correctNo, varargin) : Produces code to embed a numerical answer inside a Cloze type q
